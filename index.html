Block Elements

Block elements are elements that create a block- level box. These can be useful for developers as they can structure and organize their webpages in a visually attracting way.

Accessibility

Web accessibility means everyone, including users with disabilities, will have an ideal experience and can access the information equally.

Why is accessibility important? What are ccessibility properties like landmark roles, aria labels and image alternative texts?

accessibilit is important as it ensures all are able to access digital content. This includes people with disabilities. Landmark roles are used to define structure and purpose of a webpage. This includes headers, footers, and menus. ARIA labels are used to give additional information about elements on a webpage. Image alternative text is used to describe the content of an image for users who are unsable to see it due to visual impairments.

What is CSS?

Cascading Style Sheets is the language we use to control the appearance of our HTML pages.

What is the Difference between CSS Property and Value?

In CSS, a property is the attribute of an element that we want to change the style of. The value refers to the setting that we want to apply to that property.

Different ways to add Spacing to an Element

You can either use a margin, padding, border, line-height or flexbox.

What's the main Difference between Margin and Padding

Padding adds spacing inside the box, pushing the content away from the border. The background color is visible with the padding.

Different types of Display Properties

There are four major values for display that are used regularly: • inline — allows element on the same line; height and width properties have no effect • block — force the element to be on its own line regardless of its width • inline-block — element is formatted as an inline element but you can apply height and width values • none — the element is completely removed

Flexbox Property

Control the flow and alignment of elements for more complex layouts.

Different types of Flexbox Properties

The two types of flexbox properties are container and item properties. Container properties affect the behaviour of the flex container. Item properties affect the behaviour of the flex items.

CSS Grid Property

CSS grid is a layout that allows one to make a grid-based layout on webpages. It consitis of a parent element along with its child elements.

Parent and two Sub-properties used for CSS Grid Property

The parent property used for CSS Grid is display: grid . Two sub-properties used for CSS-grid are: 1. grid-template-colums. This is mostly to define the size and number of the columns in the grid. The other is grid-template-rows. This is used to define the size and number of rows that are in the grid.

How to add Google Fonts to your CSS File

You have to visit google fonts website and select the fonts you want to use. Then, click on "select this font" for each selected font. Once they are all selected, click on "Use" button in the bottom right corner. Then select the styles and character sets you wish to use, and copy the HTML code provided by google fonts. Paste the HTML code into the head section of your HTML file, between the opening and closing tags. In your CSS file, you can now use the font family name provided by Google fonts.

Why do we use border-box property in CSS?

The border box property is used to change the default box-sizing behaviour of HTML elements.
